1.性能监视器Stats

测试一个程序，性能上是否有瓶颈，在3D世界里，经常使用帧数的概念。 帧数：图形处理器每秒钟能够刷新几次，通常用fps（Frames Per Second）来表示。 一般为60左右。FPS表示上一秒的帧数，MS表示渲染一帧需要的毫秒数。MS越小越好,fps越大越好。

使用：

  1. npm install stats-js或引入stats.min.js
  2. 在初始化函数中new Stats()， 可通过stats.domElement.style.~修改其样式，然后通过~.appendChild(stats)将其插入到监视的区域

    如：stats.domElement.style.float = 'right';

  3. 在循环渲染处通过stats.update();更新其状态
